# M3 里程碑目标（阶段 3）

依据 `docs/QRGen.md` 第 10 节里程碑建议，M3 的目标为：

## M3：增加测试与 README 使用示例

### 目标
- 为核心功能补充自动化测试，覆盖关键路径与错误分支。
- 在 README 中补充可直接复制执行的使用示例，降低上手成本。

### 具体达成要点
**测试（必需）**
- 覆盖至少 3 类测试：
  1. 参数解析测试
  2. 生成成功路径测试
  3. 非法输入失败测试

**README 使用示例（必需）**
- 提供基础示例：
  - `qrgen "hello world"`
- 提供带输出路径示例：
  - `qrgen "https://example.com" -o ./dist/site.png`
- 提供自定义尺寸示例：
  - `qrgen "WIFI:T:WPA;S:MyWiFi;P:12345678;;" --size 512`

### 验收标准（M3）
- 自动化测试可运行，覆盖上述三类测试场景。
- README 中包含至少 3 条可运行的 CLI 使用示例。
